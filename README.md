## [已废弃]抖音爬虫抓取

------

### 项目背景
主要用于个人初学python和scrapy框架的练手项目。**该爬虫仅供学习使用**，不用做任何其他途径。

### 开发依赖

 - python3.6.1
 - scrapy1.5.0
 - mongoDB
 - APScheduler

### 项目介绍

主要通过抓取链接，获得搜索中的"热门挑战"和"热门音乐"的列表，再去通过参与人数排序拿出部分比较热门的链接抓取视频数据。分别存入mongoDB数据库中。每天凌晨更新数据。目前数据不全，没做全量的更新功能。
